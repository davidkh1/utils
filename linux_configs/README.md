# Linux Configuration Scripts

Configuration scripts and utilities for Linux systems.

## Scripts

### ubuntu_setup.sh
Script to automate the installation process on Ubuntu systems.

**Usage:**
```bash
chmod +x ubuntu_setup.sh
sudo ./ubuntu_setup.sh
```

### display_info.sh
Display system information.

**Usage:**
```bash
chmod +x display_info.sh
./display_info.sh
```
